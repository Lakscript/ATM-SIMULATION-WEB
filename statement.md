

**ATM Simulation System (Python + SQLite + Flask UI)**

---

## **Problem Statement**

Traditional command-line ATM simulations are limited, lack real data storage, and do not provide a user-friendly interface.
The objective of this project is to build an ATM system that can securely store user accounts, manage transactions, and provide a simple web-based UI for interaction.

---

## **Scope of the Project**

This project includes:

* Creating account records with token and PIN
* Handling deposits and withdrawals
* Maintaining transaction history
* Connecting Python logic to an SQLite database
* Providing a basic web interface for user interaction

**Core logic is entirely written in Python**, and **basic web pages/UI were created with assistance** to allow smoother user interaction.

---

## **Target Users**

* Students learning Python, Flask, and SQLite
* Beginners who want to understand how ATM systems work
* Anyone who wants a simple local ATM simulation for educational purposes

---

## **High-Level Features**

* Account creation with auto-generated token & PIN
* Login using token and PIN
* Deposit & withdrawal with validation
* SQLite database for account & history storage
* Transaction history with timestamps
* Simple Flask-based web UI for interaction
